# ansible-role-debian

Ansible role to manage Debian specifics


## Requirements

    None


## Dependencies

    None


## Role Variables

    None


## Example playbook

    ```yaml
    ---
    - hosts: all

      roles:
          - foolean/debian
    ```


## Supported operating systems

    * Debian (11)
    * Raspbian (11)


## Compliance

    * CIS Debian Linux 11 Benchmark v1.0.0
